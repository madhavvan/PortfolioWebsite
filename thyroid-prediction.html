Thyroid Prediction Using Advanced Statistical Methods

Introduction

Hypothyroidism is a prevalent endocrine disorder with significant health impacts. This project aims to identify key demographic and clinical indicators associated with hypothyroidism and evaluate their predictive value using statistical and machine learning techniques. The dataset, sourced from Kaggle, provides comprehensive patient medical histories and clinical measurements to enable a robust analysis.

Exploratory Data Analysis

Detailed visualizations, such as histograms and scatterplots, were used to inspect data distributions and identify outliers. Box plots revealed outliers in key variables like TSH and TT4, which were addressed using the Interquartile Range (IQR) method. Normality tests confirmed that data did not follow a normal distribution, informing the choice of statistical methods.

Figure 1: Boxplot showing outliers in TSH levels.

Statistical and Machine Learning Methods

Both logistic regression and random forest models were implemented to evaluate predictive factors for hypothyroidism. Logistic regression provided interpretable coefficients for predictors like TSH, TT4, and FTI. Random forest outperformed logistic regression in accuracy, achieving 96.02% compared to 92.57%. Correlation analysis and chi-square tests identified strong associations between key variables and hypothyroidism.

Figure 2: Correlation heatmap highlighting significant relationships between variables.

Key Results

  • Significant predictors: TSH, TT4, FTI, and T3 levels.
  • Logistic regression accuracy: 92.57%.
  • Random forest accuracy: 96.02%.
  • TSH emerged as the most influential variable for hypothyroidism prediction.

Conclusion

This project highlights the effectiveness of statistical and machine learning models in predicting thyroid disorders. By integrating these methods into clinical practice, healthcare providers can ensure early detection and improve treatment outcomes.
